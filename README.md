# Prestige Healthcare Services, Inc.

A premium, fully responsive, and performance-optimized website for **Prestige Healthcare Services, Inc.**, a licensed non-medical home care agency serving Shelby County, Tennessee.

---

## 🌟 Key Features

* **Dynamic Home Care Services Showcase**: Elegant scroll-based, hardware-accelerated stacking cards highlighting core services (Personal Care, Companionship, and Homemaker services).
* **Interactive Care Consultation Stepper**: A step-by-step user intake form with client relationship classification, Shelby County location validation, and Medicaid confirmation.
* **Interactive Map Section**: Google Maps iframe integration complete with a glassmorphism floating information card and a direct directions redirect link.
* **Header Quick-Contact (24/7 Line)**: Prominent corporate toll-free contact visibility across all viewports (desktop, tablet, and mobile).
* **Performance Optimizations**:
  * **Dynamic Responsive Video Loading**: Intercepts double video preloading by dynamically injecting/clearing standard desktop and mobile MP4 sources to save **up to 8.0 MB** of mobile bandwidth.
  * **Lazy Loading**: Native HTML5 `loading="lazy"` attributes implemented below the fold.

---

## 🛠️ Tech Stack & Structure

* **Frontend**: Pure HTML5 semantic markup, CSS3 custom variables & grid layout, and Vanilla JS ES6+ interaction scripts.
* **Typography**: Lora (headings/editorial quotes) and Cabin (UI text/body copywriting) via Google Fonts integration.
* **Icons**: Inline SVG vectors.
* **Deploy Settings**: Configured for Netlify (`netlify.toml`).
